Data contained here represent the finalized data used in for the manuscript.

"FINAL_BOTTLE" : data averaged for each bottle of each experiment. 

"FINAL_SUMMARY" : data averaged for each experiment. 

"Gaussian_Decompositions" : component Gaussian functions of unsmoothed and decomposed absorption spectra. 

"Pigment_Ratios" : peaks of individual pigment absorption Gaussian functions and ratios thereof.

"Correlations": Nonparametric Spearman correlation coefficients were used to assess relationships between phytoplankton physiological measurements and phytoplankton bio-optical properties at 470, 550, and 660 nm.

"Headers" : xlsx file containing header descriptions for each .csv
